CSS Color Properties

1. Color

  • Changes the text color of an element.
  • Example:

            p {
                color:blue;
            }

2. Background-color

  • Sets the background color of an element.
  • Example:

            body {
                background-color: lightyellow;
            }

3. Border-color

  • Defines the color of an element's border.
  • Example:

            div {
                border: 2px solid red;
            }

Ways CSS Accepts Color Values

1. Named Colors

  • Use color names that browsers recognize.
  • Example:

            h1 {
                color: coral;
            }

2. Hexadecimal Colors

  • Use a mix of 6 digits (0-9) (A-F) to define red, green, and blue values.
  • Example:

            p {
                color: #ff69b4;
            }

3. RGB Colors

  • Use numeric values for red, green, and blue (0-255).
  • Example:

            body {
                background-color: rgb(255, 228, 225);
            }

4. HSL (Hue, Saturation, Lightness)

  • Uses color values based on color tone, saturation %, and lightness %.
  • Example:

            h2 {
                color: hsl(340, 100%, 70%);
            }
