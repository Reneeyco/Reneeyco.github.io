Code Log 2: CSS Measuring Units

What is he difference between absolute and relative CSS length units?

Absolute vs. Relative CSS Length Units

  • Absolute units: Fixed sizes that don't change depending on the screen or parent element. For example, if you set something to 20px, it'll always be 20 pixels no matter what.
  • Relative units: Flexble sizes that scale based on another value, like the parent element's font size or the size of the viewport. For ecample, 2em depends on the font size of the parent, while 50% depends on the parent container's size.
  • In short: absolute = fixed, relative = flexible.
